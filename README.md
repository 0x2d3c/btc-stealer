### btc-stealer
 - 一个简单的BTC助记词碰撞示例
 - 一个不花钱的极低概率抽奖程序
### 程序构建
 - 构建环境 Golang 1.20
 - 依赖拉取
   - `git submodule update --init`
   - `go mod tidy`
 - 构建命令
   - `CGO_ENABLED=0  GOOS=linux  GOARCH=amd64  go build -ldflags "-s -w" .`
### 镜像构建
 - `sudo docker build -f Dockerfile -t btc-stealer:v1.0 `
### 程序运行
 - 直接运行
   - `btc-stealer`
 - 容器运行
   - `sudo docker run -d  btc-stealer:v1.0`
### 说明
 - ubuntu 2022.04 6C 16G 9000/s
